## Описание работы
В ходе данной лабораторной работы были реализованы 6 математических моделей с использованием линейных алгоритмов. Программы демонстрируют навыки работы с переменными, вещественными числами (double) и консольным вводом/выводом.

### Реализованные задачи:
1. **Сумма и среднее арифметическое:** расчет суммы двух чисел и их среднего значения.
2. **Геометрия прямоугольника:** вычисление площади ($S = a \times b$) и периметра ($P = 2 \times (a + b)$).
3. **Площадь круга:** использование константы $\pi$ для нахождения площади по радиусу.
4. **Конвертер температур:** перевод градусов Цельсия в Фаренгейты.
5. **Физическая задача:** вычисление пройденного расстояния по скорости и времени ($S = v \times t$).
6. **Возведение в степень:** расчет квадрата и куба введенного числа.

## Используемые технологии
* **Язык:** Java
* **Инструменты:** IntelliJ IDEA, Git, класс `java.util.Scanner`.

Каждое задание вынесено в отдельный класс в папке `src/lab3`. Для запуска откройте нужный файл и нажмите `Shift + F10`.
